When to Use Each Tool

Choose based on your specific security requirements and workflow.

Choose Auditor Core If...

  • You need complete visibility into existing security risks
  • You're conducting a security audit or due diligence
  • You need detailed reports for compliance requirements
  • You're onboarding a legacy codebase with unknown risks
  • You want to measure progress over time with baselines
Explore Auditor Core

Choose Sentinel If...

  • You need a deterministic security gate in CI/CD
  • You want to prevent specific risks from entering codebase
  • You operate in air-gapped or highly restricted environments
  • You need zero telemetry and complete privacy
  • You want to enforce team policies automatically
Explore Sentinel

Use Both For...

  • Comprehensive security program with defense in depth
  • Audit existing code while preventing new risks
  • Enterprise-grade security across SDLC
  • Regulatory compliance with enforcement
  • Maximum protection for critical infrastructure
