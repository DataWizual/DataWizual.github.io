Licensing FAQ

Can I use one license on multiple machines?

No. Each License Key is cryptographically bound to one Machine ID. A separate license is required for each machine where the product will run.

What happens if my machine changes?

If your hardware changes significantly (e.g. new server, OS reinstall that changes Machine ID), contact us. We will issue a replacement key after verification.

How long does delivery take?

Typically within 24 hours of receiving your Machine ID. You will receive the product archive and License Key via secure email.

Do I need a license for both products?

Yes. Auditor Core and Sentinel Core are licensed separately. Sentinel Core embeds Auditor Core internally — but its license is distinct and machine-specific.

Is Sentinel Core available on GitHub?

No. Sentinel Core is distributed exclusively via secure email archive. This ensures full sovereignty over your security logic and eliminates external repository dependencies.
